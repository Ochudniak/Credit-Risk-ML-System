# Credit-Risk-ML-System
EN

End-to-end machine learning system for credit default risk prediction.

DATASET

This project expects the Default of Credit Card Clients dataset.

Required raw file:
data/raw/default_of_credit_card_clients.xls

The raw dataset is not included in this repository.
Please download it from UCI Machine Learning Repository.


---------------------------------------------------------------------------------------------------------

PL

Kompleksowy system uczenia maszynowego służący do prognozowania ryzyka niewypłacalności kredytowej.

DANE

Ten projekt korzysta ze zbioru danych o nazwie „Default of Credit Card Clients”.

Wymagany plik:
data/raw/default_of_credit_card_clients.xls

Zestaw danych nie znajduje się w tym repozytorium.
Prosimy pobrać go z UCI Machine Learning Repository.
