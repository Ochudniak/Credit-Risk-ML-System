# Data Description

## Dataset name

Default of Credit Card Clients Dataset

## Source

UCI Machine Learning Repository

## Local file path

data/raw/default_of_credit_card_clients.xls

## File format

XLS file

## Problem type

Binary classification

## One row represents

One credit card client.

## Target variable

The target variable indicates whether the client defaulted on payment in the following month.

In the original dataset, the target column is named:

default payment next month

## Target classes

0 - no default  
1 - default

## Feature groups

### Credit information

- credit limit

### Demographic information

- sex
- education
- marriage
- age

### Payment history

- repayment status for previous months

### Bill statement information

- bill amounts for previous months

### Previous payment information

- payment amounts for previous months

## Important notes

The raw dataset is not committed to GitHub.  
To run the project locally, the dataset must be downloaded manually and placed in:

data/raw/default_of_credit_card_clients.xls
